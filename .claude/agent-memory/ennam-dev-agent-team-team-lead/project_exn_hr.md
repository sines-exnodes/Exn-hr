Exn-Hr is a greenfield internal HR system for a Vietnamese startup/SME (<50 employees).

**Why:** Company currently manages attendance, leave, OT, and payroll manually. Need a unified digital system.

**How to apply:**
- Three separate codebases: GoLang/Gin backend, NextJS web admin, Flutter mobile
- PostgreSQL database
- 70 tasks across 8 sprints (14-16 weeks)
- Vietnamese labor law applies: BHXH 8%, BHYT 1.5%, BHTN 1% (no TNCN)
- RBAC with 5 fixed roles: Admin, CEO, HR, Leader, Employee
- Key documents: SEED-001.yaml, REQ-001-exn-hr.md, PLAN-001-implementation.md
